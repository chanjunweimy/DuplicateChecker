# DuplicateChecker

* A program written so that the user can record entry that is unique from the existing data and prompt error when then user enter entry that is not unique.

![Screenshot of Duplicate Checker V0.1](/documentation/images/ScreenshotV0.1.JPG?raw=true "Screenshot of Duplicate Checker V0.1")

### Requirements
* It should be workable without internet.
* The data should be shared between a few computers.
